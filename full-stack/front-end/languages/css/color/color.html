CSS Color

The color property is a fundamental aspect of CSS, controlling the foreground color of text and text decorations within an element. It plays a crucial role in establishing the visual hierarchy and enhancing readability of your website.

Values
Keywords:
Predefined names like red, green, blue, etc
Hexadecimal codes:
Six-digit codes starting with a #, e.g., #ff0000 for red.
RGB/RGBA values:
Specify the red, green, and blue values directly (0-255) or with an alpha channel for transparency (0-1).
Syntax
                selector {
                    color: value;
                  }
Example
                h1 { color: red; }: Sets the color of all <h1> elements to red.
                p { color: #333; }: Sets the color of all <p> elements to a dark gray shade.
                .box { color: rgb(255, 0, 0); }: Sets the color of elements with the class .box to red using RGB values.
                a { color: rgba(0, 0, 255, 0.5); }: Makes all <a> elements blue with a 50% transparency.
