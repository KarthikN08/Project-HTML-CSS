CSS Link

The <link> tag in HTML plays a crucial role in linking your CSS styles to your web pages. It allows you to separate the presentation logic from the content, ensuring cleaner code and easier maintenance.


CSS Link Pseudo-Classes: Targeting Links by State
a:link:
This applies styles to unvisited links, meaning links the user hasn't clicked yet. It's commonly used to define the default appearance of links.
a:visited:
This targets links the user has visited at least once. It allows you to highlight visited links visually, providing a subtle indication of their interaction history.
a:hover:
This applies styles to links when the user hovers over them with their mouse cursor. It's used to provide visual feedback and indicate interactivity, encouraging users to click.
a:active:
This targets links the moment they are clicked. This allows for momentary visual changes, like a slight indentation, to confirm the click action.

Example


Code

                        a:link {
                            color: blue;
                            text-decoration: underline;
                          }
                          
                          a:visited {
                            color: purple;
                            text-decoration: none;
                          }
                          
                          a:hover {
                            color: #007bff;
                            text-decoration: underline;
                          }
                          
                          a:active {
                            background-color: #f0f0f0;
                          }
