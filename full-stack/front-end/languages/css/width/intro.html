CSS Width

The width property is a fundamental CSS property used to define the width of an element. It's a vital tool for controlling the layout and visual appearance of your web pages.

Values
Pixel (px)
Absolute unit specifying the width in pixels.
Percentage (%):
Relative unit calculating the width as a percentage of the containing block's width.
em/rem:
Relative units based on the font size of the element or the root element.
vw/vh:
Viewport units specifying the width as a percentage of the viewport width/height.
auto:
Applies the natural width of the element.
initial/inherit:
Sets the width to its initial/inherited value.
Syntax
              selector {
                width: value;
              }
Example
              h1 { width: 500px; }: Sets the width of all <h1> elements to 500 pixels.
              p { width: 75%; }: Sets the width of all <p> elements to 75% of their containing block's width.
              .box { width: 2em; }: Sets the width of elements with the class .box to twice the size of their font size.
              body { width: 80vw; }: Sets the width of the <body> element to 80% of the viewport width.
