CSS Selectors

A CSS selector is a pattern used to identify HTML elements on a web page to which specific CSS styles will be applied. In simpler terms, it's like a pointing tool that tells the browser which elements on the page should be styled in a certain way.

Types of Selectors

Element Selectors:
Target elements based on their HTML tag name.
Example: p { color: red; } selects all <p> elements and makes their text red.
CID Selectors:
Target an element with a unique ID attribute.
Example: #main-header { background-color: blue; } selects the element with the ID #main-header and sets its background color to blue.
Class Selectors:
Target elements with a specific class attribute.
Example: .highlight { font-weight: bold; } selects all elements with the class .highlight and makes their text bold.
Attribute Selectors:
Target elements based on specific attributes and values.
Examples:
a[href] selects all <a> elements with an href attribute.
img[data-type="profile"] selects all <img> elements with a data-type attribute equal to "profile".
Descendant Selectors:
Target elements within other elements.
Examples:
p > span { font-style: italic; } selects all <span> elements directly inside <p> elements and makes their text italic.
.box .content { padding: 10px; } selects all .content elements within elements with the class .box and adds padding.
Universal Selector:
Targets all elements.
Example: * { margin: 0; } sets the margin to 0 for all elements on the page.
Combinators:
Combine selectors for more specific targeting.
Commonly used combinators:
> selects a child element.
+ selects the next sibling element.
~ selects all sibling elements.
, separates multiple selectors targeting the same rule.
Pseudo-classes and Pseudo-elements:
Target elements based on their state or behavior.
Examples:
:hover targets elements when the user hovers over them.
:active targets elements when the user clicks them.
::first-line targets the first line of an element.
