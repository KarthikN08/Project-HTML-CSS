CSS Text Color

The color property is used to set the color of the text.

Values
Hexadecimal colors:
Start with #, followed by six digits representing red, green, and blue values (e.g., #ff0000 for red).
RGB values:
Specify red, green, and blue values separated by commas (e.g., rgb(255, 0, 0) for red).
HSL/HSLA values:
Use hue, saturation, and lightness/alpha to define color (e.g., hsl(0, 100%, 50%) for pure red).
Color keywords:
Predefined names like red, blue, green, black, etc.
Syntax
                selector {
                    Color: value;
                  }
Example
                p {
                    color: #ff0000; /* Red text */
                  }
                  
                  h1 {
                    color: rgb(0, 0, 255); /* Blue text */
                  }
                  
                  a {
                    color: hsl(300, 80%, 60%); /* Turquoise text */
                  }
                  
                  .highlight {
                    color: yellow; /* Using keyword */
                  }
