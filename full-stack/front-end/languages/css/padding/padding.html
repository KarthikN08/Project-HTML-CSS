CSS Padding

Padding adds a buffer zone around the content within an element, separate from the margin that exists outside the border. This empty space helps prevent text from touching the edges of the element,

Properties
Padding
This shorthand property allows you to set all sides of the padding (top, right, bottom, left) at once.
  • padding-left
  • padding-top
  • padding-right
  • padding-bottom
Value
specifies margine values.
  • Pixels (px): Precise unit for specific pixel spacing.
  • Percentage (%): Relative unit based on the element's parent element size.
  • Em (em): Relative unit based on the font size of the parent element.
  • Rem (rem): Relative unit based on the root element's font size.
Basic Syntax
                        padding: 25px 50px 75px 100px;

Example

Code
                        <h6 
                        Style="
                            margin: 100px 100px 100px 100px;
                            border: 1px solid red;
                            padding-top: 100px;
                            padding-bottom: 100px;
                            padding-right: 150px;
                            padding-left: 80px;"
                        >
                        This is Example text to display how the padding property works
                        </h6>
