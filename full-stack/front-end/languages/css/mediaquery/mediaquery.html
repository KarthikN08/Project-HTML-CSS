CSS Media Query

Media queries are a fundamental aspect of creating responsive and adaptive web designs. They enable web developers to tailor the presentation of their content based on the characteristics of the device or screen on which it is viewed.

What are Media Queries

Media queries are CSS techniques that allow you to apply styles based on certain conditions, such as screen size, resolution, or device orientation. They play a crucial role in creating websites that look great and function well across a variety of devices, from large desktop monitors to small mobile screens.

Syntax
                /* Example of a media query for screens with a maximum width of 600 pixels */
                @media only screen and (max-width: 600px) {
                  /* Styles for small screens go here */
                }

Common Media Query Features

Screen Size
                        /* Styles for screens with a width between 601px and 900px */
                        @media only screen and (min-width: 601px) and (max-width: 900px) {
                             /* Responsive styles for medium-sized screens */
                        }
Device Orientation
                        /* Styles for devices in landscape orientation */
                        @media only screen and (orientation: landscape) {
                          /* Landscape-specific styles */
                        }
Resolution
                        /* Styles for devices with a high resolution (Retina displays, for example) */
                        @media only screen and (min-resolution: 300dpi) {
                              /* High-resolution styles */
                        }
