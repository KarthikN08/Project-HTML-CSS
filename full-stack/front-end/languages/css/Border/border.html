CSS Border

CSS Borders: Adding Definition and Style to Your Webpages Borders in CSS are a powerful tool for adding visual definition, structure, and emphasis to your website elements. They can be customized in various ways to create unique and appealing designs.

Properties
border-style:
Defines the visual appearance of the border (e.g., solid, dotted, dashed, double).
  • dotted - Defines a dotted border
  • dashed - Defines a dashed border
  • solid - Defines a solid border
  • double - Defines a double border
  • groove - Defines a 3D grooved border. The effect depends on the border-color value
  • ridge - Defines a 3D ridged border. The effect depends on the border-color value
  • inset - Defines a 3D inset border. The effect depends on the border-color value
  • outset - Defines a 3D outset border. The effect depends on the border-color value
  • none - Defines no border
  • hidden - Defines a hidden border
border-width:
Specifies the thickness of the border in pixels (px).
border-color:
Sets the color of the border.
border-collapse:
Controls whether individual element borders merge or remain separate when adjoining elements have borders.
Basic Syntax
                        border: 1px solid red;

Example

Code
                        <h6 
                        Style="
                            border-top: 4px dotted blue;
                            border-right: 4px double green;
                            border-bottom: 4px dashed red;
                            border-left: 4px solid black;" 
                        >
                            This is Example
                        </h6>
