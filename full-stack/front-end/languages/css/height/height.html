CSS Height

The height property, like its companion width, is a cornerstone of CSS layout and design. It defines the height of an element and plays a crucial role in shaping the visual appearance of your web pages.

Values
Pixel (px)
Absolute unit specifying the height in pixels.
Percentage (%):
Relative unit calculating the height as a percentage of the containing block's height.
em/rem:
Relative units based on the font size of the element or the root element.
vw/vh:
Viewport units specifying the width as a percentage of the viewport width/height.
auto:
Applies the natural height of the element.
initial/inherit:
Sets the height to its initial/inherited value.
Syntax
                selector {
                    height: value;
                  }
Example
                img { height: 200px; }: Sets the height of all <img> elements to 200 pixels.
                .box { height: 50%; }: Sets the height of elements with the class .box to 50% of their containing block's height.
                p { height: 1.5em; }: Sets the height of all <p> elements to 1.5 times the size of their font size.
                #header { height: 10vh; }: Sets the height of the element with the ID #header to 10% of the viewport height.
                body { height: auto; }: Allows the <body> element to take its natural height based on its content.
