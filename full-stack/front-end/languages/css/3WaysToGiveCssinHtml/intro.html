Linking CSS to HTML

There are three primary ways to link CSS to HTML:


Inline Styles:
In this method, you directly write the CSS code within the HTML tag using the style attribute.
This is useful for quick changes or applying styles to individual elements, but not recommended for larger projects due to code duplication and maintenance issues.
Basic Syntax
                        <h1 style="color: red; font-size: 2em;">This is a heading</h1>
Result

This is a heading


Internal Styles:
This involves embedding the CSS code within the <head> section of your HTML document using the <style> element. This allows you to define styles for the entire document or specific sections without duplicating code.
Basic Syntax
                        <head>
                        <style>
                              body {
                                font-family: Arial, sans-serif;
                              }
                              h1 {
                                color: #333;
                                font-size: 2em;
                              }
                              </style>
                              </head>
External Styles:
This is the most recommended approach for larger projects. You create a separate CSS file with the extension .css and link it to your HTML document using the <link> element within the <head> section. This promotes code reusability, easier maintenance, and better organization.
Basic Syntax
                        <head>
                            <link rel="stylesheet" href="styles.css">
                          </head>
