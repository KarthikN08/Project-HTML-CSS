CSS Font

The font property in CSS allows you to control the appearance of text within an element. It encompasses several sub-properties that customize various aspects of the font, like family, size, style, and weight.

Syntax
                selector {
                    font: font-style font-variant font-weight font-size/line-height font-family;
                  }
Sub-properties:
font-style:
Normal, italic, oblique.
  • Normal: This is the default style for text and displays it upright.
  • Italic: This style slants the text to the right, often used for emphasis or stylistic purposes.
  • Oblique: This style also slants the text, but at a different angle than italics. It's less commonly used and often considered less readable than italics.
font-variant:
Normal, small-caps.
  • Normal: This is the default variant and displays the text in its standard form.
  • Small-caps: This variant displays all lowercase letters in the size of uppercase letters, keeping uppercase letters unchanged. It's primarily used for headings or short phrases.
font-weight:
Normal, bold, lighter, bolder, etc.
  • Normal: This is the default weight, displaying the text with its standard thickness.
  • Bold: This weight makes the text thicker and more prominent, often used for headings or important information.
  • Lighter: This weight makes the text thinner than the default, often used for less important text.
  • Bolder: This weight makes the text even bolder than the standard bold weight.
  • Other weights: Numbers (100 to 900) can also be used to specify specific levels of boldness, with higher numbers indicating bolder text.
font-size:
Absolute units (px, pt), relative units (em, rem), keywords (xx-small, larger).
  • Absolute units (px, pt): These units define the font size in fixed units, like pixels (px) and points (pt).
  • Relative units (em, rem): These units define the font size relative to another element. An "em" is relative to the font size of the parent element, while a "rem" is relative to the root element's font size.
  • Keywords (xx-small, larger): These keywords provide predefined relative font sizes, ranging from xx-small (smallest) to larger (largest).
    • line-height:
    Creates space between lines of text.
    font-family:
    Specifies the font family name(s).
    Example
                    p { font: italic small-caps bold 16px/1.5 serif; }: Applies various styles to <p> elements, making them italic, small caps, bold, 16px in size with 1.5 line height, and using a serif font family.
                h1 { font-size: 2em; font-family: "Open Sans", sans-serif; }: Makes all <h1> elements twice the default font size and uses the "Open Sans" font if available, fallback to a sans-serif font.
                .highlight { font-weight: bold; }: Applies bold weight to all elements with the class .highlight.
