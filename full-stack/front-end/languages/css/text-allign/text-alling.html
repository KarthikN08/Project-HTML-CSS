CSS Text Allign

Ah, text alignment! That's a key aspect of formatting your content visually and enhancing readability.

In CSS, you have several options to align your text horizontally within its element:
Basic options:
  • text-align: left; : Aligns the text to the left edge of its container. This is the default behavior.
  • text-align: center; : Aligns the text horizontally in the center of its container.
  • text-align: right; : Aligns the text to the right edge of its container.
Justified alignment:
text-align: justify; : Distributes the text evenly across the available space, creating a justified look similar to printed text. This can be helpful for large blocks of text, but be aware that it may distort spacing and introduce unwanted hyphenation.
Responsive alignment:
  • text-align: initial; : Inherits the alignment from the parent element.
  • text-align: inherit; : Same as initial, but applies to all descendants as well.
Direction-specific alignment:
  • text-align: start; : Aligns the text to the starting edge of the writing direction (left in horizontal writing, right in right-to-left writing).
  • text-align: end; : Aligns the text to the ending edge of the writing direction.
Syntax
                selector {
                    text-align: option;
                  }
Example
                .heading {
                    text-align: center;
                  }
                  
                  .paragraph {
                    text-align: justify;
                  }
                  
                  .quote {
                    text-align: right;
                    font-style: italic;
                  }
