CSS Margin

Margins are the empty spaces surrounding an element's content box. They create separation between elements, preventing them from overlapping and improving visual flow.

Properties
margin
This shorthand property allows you to set all margins at once (top, right, bottom, left) in one line.
  • margin-left
  • margine-top
  • margin-right
  • margine-bottom
Value
specifies margine values.
  • Pixels (px): Precise unit for specific pixel spacing.
  • Percentage (%): Relative unit based on the element's parent element size.
  • Em (em): Relative unit based on the font size of the parent element.
  • Rem (rem): Relative unit based on the root element's font size.
Basic Syntax
                        margin-top: 50px 50px 75px 100px;

Example

Code
                        <h6 
                        Style="
                            border: 1px solid red;
                            margin-top: 100px;
                            margin-bottom: 100px;
                            margin-right: 150px;
                            margin-left: 80px;"
                        >
                        This is Example text to display how the margin property works
                        </h6>
