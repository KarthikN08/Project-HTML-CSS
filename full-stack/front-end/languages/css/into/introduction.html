Introduction To CSS

CSS stands for Cascading Style Sheets. It's a language used to control the appearance and presentation of web pages. While HTML defines the content and structure of a web page, CSS dictates how that content is displayed visually.

CSS Basics
Syntax:
CSS rules consist of selectors (which elements are affected), properties (what to change), and values (how to change it).
Example:
h1 { color: red; font-size: 2em; } (colors <h1> elements red and makes them twice the normal font size).
Selectors:
Identify the HTML element(s) you want to style.
Properties:
Define the visual aspects to be modified (color, font, background, etc.).
Values:
Specify the desired outcome for each property.
Example


                        body {
                            background-color: lightblue;
                          }
                          
                          h1 {
                            color: white;
                            text-align: center;
                          }
                          
                          p {
                            font-family: verdana;
                            font-size: 20px;
                          }
