HTML header

The <header> tag is a semantic element introduced in HTML5 specifically used to define the header section of a web page.

using header tag effectively
Placement
Place the <header> tag after the opening <body> tag and before the main content.
Content
includes element like
  • Logo
  • site name
  • Navigation bar
  • Search bar
Nesting element
Use other HTML elements like <div>, <nav>, <ul>, <li>, <p>, and <img> within the <header> for organization and styling.
Attribute
Use the id and class attributes to identify specific elements and apply CSS styles.

Basic Syntax

                    <header>
                     page header content
                    </header>

Example

code

                        <header>
                        <nav>
                        <h1>My Website</h1>
                        <ul>
                        <li><a href="#">Home</a></li>
                        <li><a href="#">About</a></li>
                        <li><a href="#">Contact</a></li>
                        </ul>
                        </nav>
                        </header>
