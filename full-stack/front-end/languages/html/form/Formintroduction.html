HTML Forms

Forms are interactive elements used to collect user input and submit it to a server for processing. They are essential components of many web applications and can be used for various purposes, such as:

  • User registration and login: Gathering user information for account creation and authentication.
  • Contact forms: Enabling users to send messages and inquiries.
  • Surveys and polls: Collecting feedback and data from users.
  • Online orders and purchases: Processing transactions and gathering payment information.
  • Product search and filtering: Refining user searches based on specific criteria.

Basic Syntax


                    <form action="/submit" method="post">
                    <label for="name">Name:</label>
                    <input type="text" id="name" name="name" required><br>
                    <label for="email">Email:</label>
                    <input type="email" id="email" name="email" required><br>
                    <button type="submit">Submit</button>
                    </form>

Form element


<form>
Defines the main form element.
action
attribute - Specifies the URL where the form data will be submitted.
method
attribute - Defines how the form data will be sent (e.g., post or get).
<label>
Associates a label with an input field for user guidance.
<input>
Represents the input field where users enter data.
type
attribute - Defines the type of input field (e.g., text, email, password, checkbox, radio).
<button>
Represents the submit button that triggers form submission.
id
attribute - Assigns a unique identifier to the input field.
name
attribute - Identifies the name of the input field for data submission
<select>
Creates a dropdown menu for users to choose from multiple options.
<textarea>
Provides a multi-line text box for users to enter longer content.
required
attribute: Ensures users fill in the required fields before submitting the form.
Placeholder
attribute: Displays a hint inside the input field when it's empty.
disabled
ttribute: Disables an input field to prevent user interaction.

                        <input type="email" id="email" name="email" required>
                        <br>

                        <label for="url">Website URL:</label>

                        <label for="age">Age:</label>

                        <br>

                        <label for="phone">Phone Number:</label>
                        <input type="tel" id="phone" name="phone">
                        <br>

                        <label for="date">Date:</label>
                        <input type="date" id="date" name="date">
                        <br>

                        <label for="time">Time:</label>
                        <input type="time" id="time" name="time">
                        <br>

                        <label for="file">Upload a File:</label>
                        <input type="file" id="file" name="file">
                        <br>

                        <label for="message">Message:</label>
                        <textarea id="message" name="message" rows="5"></textarea>
                        <br>

                        <label>Gender:</label>
                        <br>

                        <input type="radio" id="male" name="gender" value="male">
                        <label for="male">Male</label>
                        <br>

                        <input type="radio" id="female" name="gender" value="female">
                        <label for="female">Female</label>
                        <br>

                        <label>Interests:</label>

                        <br>


                        <input type="checkbox" id="music" name="interests[]" value="music">

                        <label for="music">Music</label>

                        <br>

                        <input type="checkbox" id="reading" name="interests[]" value="reading">

                        <label for="reading">Reading</label>
                        <br>

                        <input type="checkbox" id="sports" name="interests[]" value="sports">
                        <label for="sports">Sports</label>

                        <label for="country">Country:</label>

                        <option value="uk">United Kingdom</option>
                        </select>

                        <button type="submit">Submit</button>
