HTML figure

The <figure> tag in HTML is used to represent self-contained content, often containing an image or diagram, that is related to the main flow of the document but can be moved without affecting the document's meaning.

Optional Elements:

Content:
This is typically an image (<img> tag) or a diagram (<svg> tag), but can be any element that conveys meaning.
Caption:
The figcaption element provides a textual description of the figure content, enhancing accessibility and understanding.

Attributes:


The <figure> tag also supports a few optional attributes for customization:
  • id: Assigns a unique identifier for styling or scripting purposes.
  • class: Assigns a class name for applying styles through CSS.
  • style: Specifies inline styles directly within the tag.

Basic Syntax

                    <figure>
                    <img src="" alt="">
                    <figcaption>Figure caption</figcaption>
                    </figure>

Example

code
                        <figure>
                        <img src="./img1.jpg" alt="this is image">
                        <figcaption>The subaru WRX STi</figcaption>
                        </figure>
