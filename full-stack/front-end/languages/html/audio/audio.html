HTML audio

The HTML <audio> tag allows you to embed audio files directly into your web pages, enhancing the user experience and adding multimedia elements.

Attributes:


controls: This attribute displays a built-in audio player with play, pause, volume, and seek controls.
src: This attribute specifies the URL of the audio file. You can include multiple elements with different audio formats for wider browser compatibility.
type: This attribute defines the audio file format (e.g., mp3, ogg, wav).
autoplay: This attribute automatically starts playing the audio as soon as the page loads.
loop: This attribute makes the audio loop continuously.
muted: This attribute starts the audio muted.
preload: This attribute specifies when the audio should be downloaded. Possible values include auto (download immediately), metadata (download only metadata), and none (don't download until playback).

Basic Syntax

                    <audio controls>
                    <source src="audio.mp3" type="audio/mpeg">
                    <source src="audio.ogg" type="audio/ogg">
                        Your browser does not support the audio element.
                    </audio>
