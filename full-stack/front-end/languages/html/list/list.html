HTML List

HTML lists are fundamental elements used to present information in an organized way. They come in two main types:

Ordered Lists:


  • Represented by the <ol> tag.
  • Used to present items in a specific sequence.
  • Items are numbered automatically or with custom numbering attributes.

Unordered Lists:


  • Represented by the <ul> tag.
  • Used to present items in no particular order.
  • Items are typically marked with bullets.

Attributes:


type: (for <ol>) specifies the numbering style (e.g., 1, a, i).
start: (for <ol>) defines the starting number of the list.
reversed: (for <ol>) displays the list in reverse order.
compact: (for both <ul> and <ol>) removes extra space around list items.

Basic Syntax

                        <ul>  
                        <li>List item 1</li>
                        <li>List item 2</li>
                        <li>List item 3</li>
                        </ul>
                    
                        <ol>
                        <li>Step 1</li>
                        <li>Step 2</li>
                        <li>Step 3</li>
                        </ol>

results

  • List item 1
  • List item 2
  • List item 3
  1. Step 1
  2. Step 2
  3. Step 3
