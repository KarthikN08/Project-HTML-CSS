HTML aside

In HTML, <aside> is a semantic tag used to define content that is related to the main content but is considered separate and tangential. It can be used for sidebars, callouts, quotes, or other content that provides additional information without interrupting the main flow.

Basic Syntax
                    <aside>
                        The article content
                    </aside>

Example

code 

                        <aside>
                        <p>The Epcot center is a theme park at Walt Disney World Resort featuring 
                                exciting attractions, international pavilions, award-winning fireworks and seasonal special events.</p>
                         </aside>

Results

The Epcot center is a theme park at Walt Disney World Resort featuring exciting attractions, international pavilions, award-winning fireworks and seasonal special events.
