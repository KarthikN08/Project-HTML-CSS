HTML iframe

An iframe, or inline frame, is an HTML element used to embed another HTML page within the current page.
It essentially acts like a window within your webpage, displaying the content of another webpage.

Uses of iframes

Iframes have a variety of uses, including:
Embedding widgets: You can use iframes to embed social media widgets, like a Twitter feed or Facebook comments section.
Embedding ads: Many websites use iframes to display ads from third-party networks.
Embedding video players: You can use iframes to embed videos from YouTube or other video platforms.
Embedding interactive content: You can use iframes to embed interactive elements, such as games or maps.
Creating a consistent header or footer: You can use iframes to create a header or footer that appears on all your pages.

attributes

src: This attribute specifies the URL of the content to be embedded. This can be a full URL to another website or a relative path to a resource on your own server.
width: This attribute specifies the width of the iframe in pixels.
height: This attribute specifies the height of the iframe in pixels.
frameborder: This attribute controls whether or not a border is displayed around the iframe.
scrolling: This attribute determines whether scrollbars are displayed within the iframe.
sandbox: This attribute provides a security sandbox to restrict the capabilities of the embedded content.
allowfullscreen: This attribute allows the embedded content to enter fullscreen mode.

ifrmae example


<iframe src="URL of the content to embed" width="width" height="height"></iframe>
