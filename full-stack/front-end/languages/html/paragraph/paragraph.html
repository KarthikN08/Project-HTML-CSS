HTML Paragraph

The HTML paragraph element, represented by the <p> tag, defines a paragraph of text. It is a fundamental building block of any HTML document and helps in structuring and organizing content.

Attributes:

While the <p> tag can be used with no additional attributes, it supports a few options for customization:
align: This attribute specifies the horizontal alignment of the paragraph text. Possible values include left (default), center, right, and justify.
id: This attribute assigns a unique identifier to the paragraph element, useful for styling or scripting purposes.
class: This attribute allows assigning a class name to the paragraph element for applying styles through CSS.

Basic Synatx


                    <p>This is a paragraph.</p>
