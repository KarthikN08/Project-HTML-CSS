HTML video tag

The <video> element is a powerful tool in HTML that allows you to embed video content directly into your web pages. This opens up a world of possibilities for adding multimedia experiences to your websites.

attributes used in video tag

src: This attribute specifies the path to the video file.
controls: This attribute adds playback controls to the video, such as play, pause, and volume.
width: This attribute specifies the width of the video in pixels.
height: This attribute specifies the height of the video in pixels.

img tag syntax

                    <video src="path/to/video.mp4" controls width="640" height="360"></video>
