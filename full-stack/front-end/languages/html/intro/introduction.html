Introduction to HTML

HTML, or HyperText Markup Language, is the standard markup language for creating web pages. It is used to define the structure and content of a web page, including headings, paragraphs, images, links, and forms. HTML is a relatively simple language that can be learned by anyone with a basic understanding of computers.

Here are some of the basic building blocks of HTML:

Tags: Tags are the basic building blocks of HTML. They are used to mark up the content of a web page. Tags are enclosed in angle brackets (<>) and have a name and an optional set of attributes. For example, the following tag marks up a heading:
<h1>This is a heading</h1>

Elements: An element is a combination of a tag and its content. The content of an element is the text or other elements that are enclosed within the tag. For example, the following element is a heading that contains the text "This is a heading":
<h1>This is a heading</h1>

Attributes: Attributes are used to provide additional information about an element. Attributes are specified within the tag's opening angle bracket and consist of a name and a value. For example, the following tag marks up an image with the source attribute set to "image.jpg":
<img src="image.jpg" alt="Image">

HTML INLINE and BLOCK element

In HTML, elements are categorized into two main types: inline elements and block elements. These categories define how elements are displayed within the document flow and how they interact with surrounding content.

Inline Elements:

Flow horizontally within the line of text.
Do not occupy the full width available.
Examples: span, a, strong, em, img, input, button, br.

Block Elements:

Start on a new line and occupy the full width available.
Create separate sections or blocks of content.
Examples: p, div, h1-h6, ul, ol, table, form.

Example of HTML structure

<!DOCTYPE html>
<html>
<head>
<title>My Web Page</title>
</head>
<body>
<h1>This is a heading</h1>
<p>This is a paragraph of text.</p>
<img src="image.jpg" alt="Image">
<a href="https://www.example.com">Link</a>
</body>
</html>
