HTML Footer

The <footer> tag is used to define the footer section of the webpage.

Attributes

Use the id and class attributes to identify specific elements and apply CSS styles.

Basic Syntax

                    <article>
                    The article content
                    </article>

Example


Code

                        <footer>
                        <p>Author: Hege Refsnes</p>
                        <p><a href="mailto:hege@example.com">hege@example.com</a></p>
                        </footer>
