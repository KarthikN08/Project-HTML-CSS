HTML Time

The <time> tag in HTML defines a specific point in time or a duration. It's a semantic element that provides both human-readable and machine-readable information about time, aiding accessibility and machine parsing.

Purpose
  • Represent a specific date and/or time.
  • Indicate event durations or deadlines
  • Provide temporal information for search engines and assistive technologies.

Attribute

datetime
Mandatory attribute specifying the date and time in a machine-readable format (e.g., 2023-12-11T18:25:00Z for 6:25 PM PST).
pubdate
Indicates the publication date of the content.
title
Provides an optional title for the time information.

Basic Syntax


                    <time datetime="2008-01-01 24:00">New year</time>

Example

Code

                        <time datetime="2024-01-15">Submission deadline: January 15, 2024</time>

Results

Submission deadline: January 15, 2024
