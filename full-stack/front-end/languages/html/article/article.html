HTML Article

The <article> tag is an important element in HTML5 for structuring and defining self-contained content. It allows you to clearly indicate independent sections within a document, enhancing accessibility and improving the overall organization of your website.

Purpose
  • To define an independent, self-contained piece of content.
  • Content within an <article> should be meaningful on its own.
  • Used for blog posts, news articles, forum posts, product descriptions, etc.

Attribute

id
Assigns a unique identifier for styling or scripting purposes.
class
Applies a class name for styling with CSS
style
Specifies inline styles directly within the tag.

Basic Syntax


                    <article>

                    </article>

Example

Code

                        <article>
                        <h2>Google Chrome</h2>
                        <p>Google Chrome is a web browser developed by Google, released in 2008. 
                        Chrome is the world's most popular web browser today!</p>
                        </article>
