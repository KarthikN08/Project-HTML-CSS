HTML image

The img tag is used to embed an image in an HTML document. It's a simple but essential element for displaying visual content on your web pages.

Attributes:


src: This attribute is required and specifies the URL of the image file. The image can be located on the same server as the HTML document, or it can be hosted on a different server.
alt: This attribute is also required and specifies alternative text for the image. This text is displayed if the image cannot be loaded, or if the user is using a screen reader.
width: This attribute specifies the width of the image in pixels.
height: This attribute specifies the height of the image in pixels.
title: This attribute specifies a tooltip that appears when the user hovers over the image with their mouse.
border: This attribute specifies the thickness of the border around the image.
align: This attribute specifies the alignment of the image relative to the surrounding text.
hspace: This attribute specifies the horizontal margin (in pixels) around the image.
vspace: This attribute specifies the vertical margin (in pixels) around the image.

image tag syntax


                    <img src="url " width="width" height="height" alt="alternative text">
