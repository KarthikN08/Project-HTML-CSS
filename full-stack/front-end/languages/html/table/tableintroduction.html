HTML Table

Tables are used to organize and display data in a structured way with rows and columns. They can be a powerful tool for presenting information in a clear and concise manner.

Elements:

<table>
Defines the main table element.
<tr>
Defines a table row.
<th>
Defines a table header cell.
<td>
Defines a table data cell.

Attributes

  • id: Assigns a unique identifier to the table for styling or scripting purposes.
  • class: Assigns a class name to the table for applying styles through CSS.
  • border: Specifies the border thickness of the table.
  • cellpadding: Adds padding around the content within each cell.
  • cellspacing: Adds spacing between individual cells.

Basic Syntax

                    <table>
                     <tr>
                     <th>Column 1 </th>
                     <th>Column 2 </th>
                     </tr>
                     <tr>
                     <td>Data 1</td>
                     <td>Data 2</td>
                     </tr>
                     </table>
